### Amazon SNS의 주요 구성 요소
-----
1. 토픽 (Topic) : SNS의 커뮤니케이션 채널
2. 구독 (Subscription) : 토픽으로 들어온 메세지를 받아볼 수 있는 리소스
3. Publisher : 메세지를 생산하고 SNS에 전달하는 주체
4. Subscriber : 실제 구독으로부터 메세지를 받아서 처리하는 주체
5. 메세지 : SNS에서 전달하는 데이터
6. 액세스 정책 : SNS에 접근할 수 있는 주체에 대한 권한 설정 정책
