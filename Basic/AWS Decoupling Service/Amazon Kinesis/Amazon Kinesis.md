-----
### Amazon Kinesis
-----
1. 실시간 스트리밍 데이터를 손쉽게 수집, 처리 및 분석할 수 있으므로 적시에 통찰력을 확보하고 새로운 정보에 신속하게 대응할 수 있음
2. 모든 규모의 스트리밍 데이터를 비용 효율적으로 처리할 수 있는 핵심 기능과 더불어 애플리케이션 요구 사항에 가장 적합한 도구를 선택할 수 있는 유연성 제공
3. AWS에서 스트리밍 데이터를 수집하고 분석 처리하는 서비스
4. 총 4가지 서비스
   - Kinesis Data Stream : 스트리밍 데이터를 수집하는 서비스
   - Amazon Data Fibrehouse (이전 : Kinesis Data Firehose) : 데이터를 수집하고 변환하여 S3 공급자 (S3, RedShift, AWS OpenSearch, DataDog, HTTP 등)에게 전송하는 서비스
   - Kinesis Data Analytics Studio : 실시간 스트리밍 데이터를 분석하는 서비스
   - Kinesis Videos Streams : 재생, 분석 및 기계 학습을 위해 미디어 스트림을 캡쳐, 처리 및 저장
