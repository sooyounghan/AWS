### 관계형 데이터베이스
-----
1. 데이터의 관계에 집중한 데이터베이스 : 사전에 정의된 관계가 있을 때 사용
2. 미리 지정된 형식과 타입의 데이터만 저장 가능
3. 테이블의 형식으로 데이터를 관리 : 행과 열을 기반으로 한 여러 테이블을 통해 데이터를 정의
4. 고유의 키로 각 데이터를 식별
5. 트랜잭션 지원
   - 원하는 동작이 정확히 수행되거나 완전히 실패 둘 중 하나로 유지
   - 즉, 어중간한 상태(일부만 반영 또는 실패했으나 데이터 변경)는 없음
6. 사용 사례 : 일반적인 애플리케이션, 온라인 게임 등
