### S3 객체 잠금
-----
1. Write Once, Read Many(WORM) 모델을 활용해 객체를 저장
2. 고정된 시간, 혹은 무기한으로 객체의 삭제 / 덮어쓰기 방지 가능
3. 규정 준수 및 객체의 보호를 위해 사용
4. 버전의 활성화 필요
5. 종류와 종류에 따른 두 가지 모드
   - 보관 모드 (Retention Mode) : 일정 기간 동안 수정 방지
     + 규정 준수 모드 : 누구도 잠금 설정 변경, 객체 설정 불가능
     + 거버넌스 모드 : 특별 권한 없이 삭제 혹은 잠금 설정 변경 불가능
       * 객체 삭제 방지 혹은 규정에 따라 보관하기 위해 사용
       * 규정 준수 모드 테스트
   - 법적 보존 (Legal Hold)
     + Hold를 객체에 부여하고, Hold가 존재하는 한 객체 삭제, 수정 불가능
     + 제한 기간이 없음
